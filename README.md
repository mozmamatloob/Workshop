I’m excited to share my latest data visualization project built using HTML, CSS, JavaScript, and Chart.js, where I created an interactive storytelling dashboard based on global development data from the World Bank.

What this project covers:
GDP Per Capita distribution & outlier analysis
Time-series trends of global GDP vs Internet usage
Correlation heatmap of socio-economic indicators
Scatter analysis: Internet access vs Economic prosperity
Life expectancy comparison across income groups

Key Highlights:
 Story-based step navigation
 Dark-themed modern UI
 Interactive charts & dynamic rendering
 JSON-driven data pipeline
This project helped me strengthen my skills in front-end development, data visualization, and exploratory data analysis, while transforming raw data into meaningful insights.
